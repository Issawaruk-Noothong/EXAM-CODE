Web Programming

HTML (Hypertext Markup Language) เป็นภาษาคอมพิวเตอร์ในรูปแบบหนึ่งที่ใช้สำหรับสร้างเว็บ (Web Page) เพื่อเก็บข่าวสารข้อมูลที่ต้องการในรูปของเอกสารไฮเปอร์เท็กซ์ (Hypertext) ที่ถูกพัฒนาโดย Tim Berners Lee โดยมีส่วนประกอบ <HTML> และ </HTML> เป็นตัวกำหนดขอบเขตไฟล์ซึ่งหัวเรื่องมีไว้กำหนดของหน้าเว็บ เช่น ชื่อเรื่องของเว็บภายใน Tag และ </HEAD> และสำหรับส่วนเนื้อหามีไว้กำหนดรายระเอียดต่างๆที่ต้องการแสดงบนเว็บเช่น ข้อความ และรูปภาพภายใน TAG </BODY> และ <BODY>

สิ่งที่ต้องรู้
- มีนามสกุลไฟล์ .html
- HTML เป็นภาษา ASCII (แอสกี)
- โครงสร้าง HTML แบ่งออกเป็น 2 ส่วน คือ ส่วนหัว (Head Section) และส่วนเนื้อหา (Body Section)
- HTML5 คือ รุ่นล่าสุด

คำสั่งต่างๆที่จำเป็นต้องรู้
<Center> จัดให้อยู่กึ่งกลาง
ตัวอย่างการใช้ <Center>Hello World</Center>
<IMG> แทรกรูปภาพ
ตัวอย่างการใช้ <IMG SRC="images/kb_rgb.gif">

<U> ขีดเส้นใต้

JAPAN
<I> ตัวเอียง ตัวอย่างการใช้ <I>Text</I>
Japan

Google ตัวอย่างการใช้ <A href="URL">Your Link</A>
